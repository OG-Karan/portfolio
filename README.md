# Built By Krn Portfolio

Minimal warm premium founder portfolio for Karan Singh Bhati / Built By Krn.

## Hero Video

Place the intro video here:

```text
public/videos/intro-video.mp4
```

The intro appears first on every refresh, plays once, and fades into the portfolio when it ends or when Skip Intro is clicked.

Place the sample or final fullscreen hero video here:

```text
public/videos/hero-sample.mp4
```

If the video file is missing, the page gracefully falls back to the warm gradient hero background. The video component already includes autoplay, muted playback, looping, `playsinline`, and bottom-right play/mute controls.

## Run Locally

Open `index.html` directly in a browser, or serve the folder locally:

```bash
python -m http.server 4173
```

Then visit:

```text
http://127.0.0.1:4173/
```
